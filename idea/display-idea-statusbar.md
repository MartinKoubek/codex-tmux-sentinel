# Display Idea In Status Bar

## Idea

When an idea note is created in the project, the tmux plugin should display that idea in the tmux status bar so it is visible while working in the window.

## Initial Shape

- Goal: Surface the current project idea directly in tmux instead of requiring the user to open the note file.
- Useful next step: Decide which idea file is considered active, then extend the status-bar helper to read and render its title or summary.
